Board Sync

Sync is automatic — nothing to set up. Changes save on this computer instantly and are pushed to a free cloud sync channel a few seconds later; every menu board display checks that channel and updates itself within about half a minute. The channel id is part of each display URL, so just copy the URLs from this admin onto your TVs.
